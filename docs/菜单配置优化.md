# MinIO组件菜单配置文档

## 问题说明

MinIO组件在侧边栏出现多余的"文件管理器"菜单项。

## 解决方案

在 `MinioApp/Bucket.xml` 中设置 `menu-include="false"`：

```xml
<subscreens-item name="FileExplorer"
                 location="component://moqui-minio/screen/MinioApp/Bucket/FileExplorer.xml"
                 menu-title="文件管理器"
                 menu-index="2"
                 menu-include="false"/>
```

## 配置说明

- `menu-include="false"`: 阻止子屏幕出现在导航菜单中
- 功能仍然可用，只是不在菜单中显示

## 应用自动发现

MinIO组件通过以下配置实现自动发现：

```xml
<screen default-menu-title="对象存储"
        menu-image="fa fa-database"
        menu-image-type="icon">
```
